## 三、发现的问题

### 🔴 高风险: 0 个

### 🟡 中风险: 2 个

| # | 问题 | 位置 | 建议 |
|---|------|------|------|
| 1 | 历史记录 API 无认证保护 | `/history/list`, `/history/download`, `/history/delete` | 添加 `@login_required` 或 `@optional_login`，确保用户只能查看自己的记录 |
| 2 | 退款回调验证逻辑待确认 | `payment_service.py` | 确认 `verify_refund_callback` 是否正确验证微信退款回调签名 |

### 🟢 低风险: 1 个

| # | 问题 | 位置 | 建议 |
|---|------|------|------|
| 1 | 缺少全局错误处理器 | `api.py` | 添加 `@api_bp.errorhandler(500)` 统一处理异常，避免暴露堆栈信息 |
