### 2.1 充值资金流向
```
用户微信支付 → 微信商户号 (1105125033) → 提现到对公账户/法人银行卡
```

**配置信息**:
- 商户号：`1105125033`
- 回调 URL：`https://xn--gmq63iba0780e.com/api/recharge/callback/wechat`
- 证书路径：`/etc/hairstyle_transfer/certs/wechat/`

**资金流程**:
1. 用户发起充值请求
2. 后端创建充值订单（`RechargeRecord`）
3. 调用微信支付 JSAPI
4. 用户完成支付
5. 微信回调后端接口
6. 验证签名并更新订单状态
7. 增加用户头发丝余额
8. 资金进入微信商户号
9. 商户号可手动提现到绑定的银行账户
