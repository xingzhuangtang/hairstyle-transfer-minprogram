## 📋 小程序发布流程

### 步骤 1: 代码上传

1. 打开微信开发者工具
2. 点击右上角"上传"按钮
3. 填写版本号和备注
4. 等待上传完成

### 步骤 2: 提交审核

1. 登录微信公众平台
2. 进入 版本管理
3. 选择刚上传的版本
4. 点击"提交审核"
5. 填写审核信息：
   - 功能页面：首页、个人中心、充值页面等
   - 测试账号：提供测试账号（如需）
   - 补充说明：AI 发型迁移工具

### 步骤 3: 等待审核

- **审核时间**: 通常 1-3 个工作日
- **审核状态查询**: 版本管理 → 审核版本

### 步骤 4: 发布上线

1. 审核通过后，收到微信通知
2. 登录微信公众平台
3. 进入 版本管理 → 审核版本
4. 点击"发布"
5. 确认发布

## 🚨 应急预案

### 问题排查

| 问题 | 检查项 | 解决方法 |
|------|--------|---------|
| 小程序无法访问 | 服务器状态、域名配置 | 检查服务运行、域名配置 |
| 支付失败 | 证书、回调地址 | 检查证书权限、回调 URL |
| 登录失败 | 微信配置、JWT 密钥 | 检查 app_id、secret |
| AI 处理失败 | 阿里云 API、账户余额 | 检查 API Key、账户余额 |

### 回滚方案

如需回滚到旧版本：
1. 微信公众平台 → 版本管理
2. 选择上一个稳定版本
3. 点击"回滚"
